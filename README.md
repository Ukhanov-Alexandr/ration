# «Мой рацион» — приложение о здоровом питании

Справочник продуктов с БЖУ, рецепты, дневник калорий, план на неделю, список покупок,
профиль с расчётом нормы калорий и обучающие материалы. Интерфейс на русском.

Статический сайт на vanilla JS: без сборки, без npm, без фреймворков. Бэкенда нет —
данные живут в браузере пользователя (`localStorage` + `IndexedDB` для фото рецептов),
резервная копия выгружается и загружается json-файлом.

## Запуск

Открыть `index.html` в браузере или поднять любой статик-сервер из корня:

```bash
python -m http.server   # → http://localhost:8000
```

Сборки и тестов нет, проверка ручная.

## Установка на телефон

Открыть сайт и добавить на домашний экран. На iOS это заодно защищает данные:
у веб-приложения с домашнего экрана своё хранилище, которое Safari не очищает
после недели простоя, — в отличие от обычной вкладки.

Хранилище у вкладки и у приложения с экрана **разное**, поэтому ставить на экран
нужно до того, как вносить данные или импортировать резервную копию.
